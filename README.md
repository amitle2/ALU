# ALU

An 4-bit arithmetic-logic unit written in Verilog, Consists of 4_1Mux, 4Bit Adder\Substractor and Full Adders.


The ALU performs arithmetic operations between 2 4-bit variables.

## The ALU operates in 4 modes:

1.NAND

2.Substract

3.NOR

4.Add
